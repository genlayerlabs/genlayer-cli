---
title: staking validators
---

List validators with stake, status, and optional explorer performance

### Usage

`$ genlayer staking validators [options]`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
|  | --all | Include banned validators | No |  |
|  | --json | Output machine-readable JSON | No |  |
|  | --sort-by &lt;field&gt; | Sort validators by stake or uptime (default: stake) | No | `stake` |
|  | --explorer-url &lt;url&gt; | Explorer backend or explorer base URL for optional performance enrichment | No |  |
|  | --network &lt;network&gt; | built-in or custom network alias (see: genlayer network list) | No |  |
|  | --rpc &lt;rpcUrl&gt; | RPC URL for the network | No |  |
|  | --staking-address &lt;address&gt; | Staking contract address (overrides chain config) | No |  |
| -h | --help | display help for command | No |  |
