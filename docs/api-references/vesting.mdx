Vesting operations for beneficiaries

### Usage

`$ genlayer vesting [options] [command]`

### Arguments

- `[command]`

### Options

| Short | Long | Description | Required | Default |
| --- | --- | --- | :---: | --- |
| -h | --help | display help for command | No |  |

### Subcommands

- `genlayer list` — List beneficiary vesting contracts and state
- `genlayer delegate` — Delegate vesting-held tokens to a validator
- `genlayer undelegate` — Undelegate all vesting delegation shares
- `genlayer claim` — Claim vesting delegation withdrawals after
- `genlayer withdraw` — Withdraw vested tokens to the beneficiary
- `genlayer validator` — Vesting-backed validator operations
